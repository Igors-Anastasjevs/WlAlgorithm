This repository is for implementation Weisfeiler-Lehman algorithm in 2 ways in WLalg.py and WlalgShevashidze.py.
main.py is file for manipulatingwith functions in WLalg.py and WLalgShevashidze.py files.
savedata.py is for saving in xlsx files.
tests.py is set of unit tests.

Before launching anything in this repository, "pip3 install -r requirements.txt" is required to be launched.
Python 3.8 is recommended, as it was written on this version.
Unit tests are launched by command "python tests.py"
